## 廃止された機能 {#features-weve-phased-out}

**更新をフィルター**：エクスペリエンスを合理化するために、以下のようにフィルターの選択を調整しました。

* **日付タイプ、指標、Dimension、グループ化**：これらの設定は、各ダッシュボードに対して事前に定義され、インサイト配信を最適化するようになりました。 詳細は、ツールチップおよび付属のドキュメントからアクセスできます。
* **キャンペーンのソース**
* **アカウント ID と名前**：レビュー中。
