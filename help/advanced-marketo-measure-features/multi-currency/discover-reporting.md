# Discover レポート {#discover-reporting}

内 [!UICONTROL ユーザー設定]を使用すると、ユーザーはデフォルトの通貨から、表示されている会社の通貨に通貨を変更することができます。 これにより、3 文字の通貨コードと共に、例えばコスト、売上高、パイプライン売上高などの金額の表示が変更されます。
